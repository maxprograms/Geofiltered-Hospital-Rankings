# Geofiltered-Hospital-Rankings

How to load Medmetrics

1) After downloading and loading the files, load the cmake file to build the project by right clicking on the cmake file and   clicking load cmake project
2) run main.cpp in the src folder
3) Press 1-8 to go to the different options available.
4) Note that options 3-5 needs option 1 to be completed for functionality
5) In option 1, input a valid city name that exists in the US alongside its corresponding state initial (ex. Gainesville | FL). Then input a valid maximum distance, pressing enter after inputting a valid input for each input box. When everything is correct, you will see a screen that says "Location and radius set successfully!" Press ESC to go back to the main menu.
6) In option 2, the user clicks on each numbered category and types in a number from 1-5, pressing enter when satisfied. Press ESC to return to the main menu.
7) In option 3, the program displays a list of the top hospitals based on the user's inputs. In this option, the user can select 1-n (where n is the lowest ranked number on the table) and doing so will display a message that says "Hospital ranked x has been selected. This option will also show the runtime for the selected data structure in searching for the hospitals. Press ESC to return to main menu
8) In option 4, the user can see the selected hospital's (from option 3) details. Press ESC to return to main menu.
9) In option 5, the user can enter in "Y" or "y" to create a results.txt file located in cmake-build-debug folder under the main folder Geofiltered-Hospital-Rankings-main. Press esc in the program to return to the main menu. 
10) In option 6, the user can press 1 and enter to select the geohash data structure for seaarching for hospitals or press 2 for a quadtree data structure for searching for hospitals. Press esc to return to the main menu.
11) In option 7, the user can see the details of the program. Press esc to return to the main menu.
12) In option 8, the user can close the program
